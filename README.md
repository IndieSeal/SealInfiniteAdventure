# Sealie's Infinite Adventure
Help a poor seal who traveled far away get back home, this is a very silly prototype of what I would like to make a full game!<br><br>
Hold the Space key to start moving upwards and drop the key to move downwards, avoid obstacles such as pufferfishes and your biggest predator... the orca. But that isn't your biggest enemy, <b>IT IS I, YOUR STOMACH!</b> You need to eat fish and squids to recover hunger, if you don't anything for too long, you might pass out.<br>

This project was made for the OneKey Program from Hack Club, in this case you only have to press the 'Space' key to move. I had originally started this project as a platformer infinite runner, like the Dino Chrome Game, but the twist was that if you held the button you could fly, which after some hours of trying to make it work, I backed down and decided to just make it so you just swam, and I really liked the loop of that and kept that idea!

This is a game made with the Unity Game Engine with the C# Language and took me about 27 hours of creation and I plan on expanding the project for publishing on mobile devices later on!
